Getting Started

Accessing the WMS

Open the web browser on your TC27 device and navigate to:

warehouse.craftculture.xyz

Log in with your admin credentials to access the WMS dashboard.

Equipment

Device Purpose
Zebra TC27 Mobile scanner for warehouse operations
Zebra ZD421 Label printer (4x2" case labels, location labels)

Barcode Formats

Type Format Example

Location Label LOC-{aisle}-{bay}-{level} LOC-A-01-02
Batch Label BATCH-{year}-{sequence} BATCH-2026-0042
The TC27's built-in scanner acts as a keyboard. Point, scan, and the barcode will type automatically into any input field.

Receiving Shipments

When goods arrive at the warehouse, use this workflow to receive them into stock.

Before You Start

  • Verify the shipment number matches paperwork
  • Check that the shipment appears in "Pending Shipments" list
  • Have the ZD421 printer ready with labels loaded

Step-by-Step Process

  1. Select Shipment
    From the WMS menu, tap Receive. Find your shipment in the list. The card shows shipment number, partner, and total cases.
  2. View Products
    Tap the shipment to see all products. Each card shows: Product Name, Vintage, Pack Size (e.g., 6x750ml), and Owner.
  3. Select Product
    Tap a product to open its detail view. The screen will scroll to the top automatically.
  4. Verify Quantity
    Check the expected quantity matches what you received. Adjust if different (e.g., received 10 cases instead of 12).
  5. Download Labels
    Tap Download Labels. A .zpl file will download. Open it with the Printer Setup app to print.
  6. Scan Bay Location
    The scan input is automatically focused. Scan the bay barcode where you will put the cases.
  7. Confirm and Continue
    Tap Confirm to save this product. Move to the next product and repeat.
  8. Complete Receiving
    After all products are processed, tap Complete Receiving. Stock is now in the system.
If you need to stop mid-way, your progress is saved automatically. Return to the shipment later to continue.
If you receive a different pack size than expected (e.g., ordered 12x750ml but received 6x750ml), adjust the pack configuration when entering quantities.

Transfer Stock

Move cases from one location to another within the warehouse.

When to Transfer

  • Reorganizing stock for better warehouse layout
  • Moving stock to make room for new arrivals
  • Consolidating partial cases in one location

Step-by-Step Process

  1. Open Transfer
    From the WMS menu, tap Transfer.
  2. Scan Source
    Scan the case barcode or location barcode where stock is currently stored.
  3. Select Quantity
    If moving partial quantity, enter how many cases to transfer.
  4. Scan Destination
    Go to the new location and scan its barcode.
  5. Confirm Transfer
    Verify the information and tap Confirm.
Stock is assigned to locations during Receiving. Use Transfer only when you need to move stock after it has been received.

Picking Orders

Pick cases from storage to fulfill customer orders.

How Pick Lists Are Created

Pick lists are created by the admin team from customer orders. You'll see them appear in your Pick Lists queue.

Pick List States

Status Meaning
Pending Ready to be picked, not started
In Progress Currently being picked
Completed All items picked

Step-by-Step Process

  1. Select Pick List
    From WMS menu, tap Pick. Find and tap your assigned pick list.
  2. Go to Location
    The first item shows the location code. Go to that bay.
  3. Scan Location
    Scan the location barcode to confirm you're at the right place.
  4. Scan Case
    Scan the case barcode to confirm you have the right product.
  5. Confirm Quantity
    Enter the quantity you're picking and tap Confirm.
  6. Repeat
    Continue for each item on the pick list.
  7. Complete Pick List
    When all items are picked, tap Complete. Download labels if needed.
If stock is not where expected, check other locations for the same product. Report any discrepancies to admin.

Repacking Cases

Split larger cases into smaller configurations (e.g., 12-pack into two 6-packs).

When to Repack

  • Customer orders a quantity that requires splitting a case
  • Converting bulk stock into retail-ready packs
  • Changing pack configuration for distribution

7-Step Repack Process

  1. Scan Source Bay
    Go to the bay containing the case to repack. Scan the bay barcode.
  2. Select Case
    Choose which stock record to repack from the list.
  3. Select Target Size
    Choose the new case configuration (e.g., 6-pack, 3-pack, 2-pack).
  4. Remove Case
    Physically take the case from the shelf. Tap Confirm Removed.
  5. Perform Repack
    Open the case and repack bottles into new case(s). Follow on-screen instructions.
  6. Scan Destination Bay
    Scan the bay where you will store the repacked cases. Can be same or different location.
  7. Download Labels
    Print new labels for the repacked cases.
The source location and destination can be different. This is useful when reorganizing stock.

Example

SOURCE: 1x 12-pack (12 bottles) ↓ Repack ↓ TARGET: 2x 6-pack (6 bottles each) New labels generated for both 6-packs

Dispatch Batching

Group picked orders into batches for efficient delivery.

Batch Workflow

Create Batch → Add Orders → Picking → Staged → Dispatched → Delivered

Step-by-Step Process

  1. Create Batch
    From WMS menu, tap Dispatch then New Batch. Select the distributor.
  2. Add Orders
    Add confirmed orders to the batch. Orders must be picked first.
  3. Generate Pick Lists
    If orders aren't picked yet, create pick lists for the batch.
  4. Stage Orders
    When picking is complete, mark the batch as Staged.
  5. Print Documents
    Download delivery note PDF and case labels for the batch.
  6. Dispatch
    When goods leave the warehouse, mark as Dispatched.
  7. Confirm Delivery
    When customer confirms receipt, mark as Delivered.

Label Printing

Print labels for cases, locations, and bay signage.

Label Types

Type Size Use
Case Label 4" x 2" Individual wine cases (barcode)
Location Label 4" x 2" Rack and bin locations (QR code)
Bay Totem 4" x 6" Multi-level bay signage (4 QR codes)

Printing Workflow

  1. Download Labels
    Click the Download Labels button in any WMS workflow.
  2. Open File
    A .zpl file will download. Tap to open it.
  3. Select Printer App
    Choose Printer Setup Utility to open the file.
  4. Print
    The app will send the label to the ZD421 printer.

Troubleshooting

Scanner Issues

Barcode won't scan

  • Ensure the barcode is not damaged or obscured
  • Hold scanner 6-10 inches from barcode
  • Try scanning at a slight angle to avoid glare
  • If still not working, type the barcode manually

Wrong barcode scanned

  • Wait for error message to clear (about 2 seconds)
  • Scan the correct barcode

Printer Issues

Labels not printing

  1. Check printer is turned on and has paper
  2. Ensure Bluetooth is connected (blue light on printer)
  3. Try downloading the label file again
  4. Restart the Printer Setup app

Labels printing incorrectly

  • Verify correct label stock is loaded (4x2" for cases)
  • Run a calibration on the printer
  • Check for paper jams or misalignment

Stock Discrepancies

Stock not where expected

  1. Check nearby locations for the product
  2. Search by product name in Stock view
  3. Report discrepancy to admin for investigation

Quantity doesn't match

  • Count physical cases again
  • Check if cases were moved or picked
  • Report to admin - may need stock reconciliation

App Issues

Screen not responding

  • Wait a few seconds - may be loading
  • Refresh the page (pull down or tap refresh)
  • If stuck, close and reopen the browser

Keyboard keeps appearing

  • Tap outside the input field to dismiss
  • The scan input is designed to not show keyboard
  • If persistent, refresh the page
If you encounter a critical error that prevents work, contact the admin team immediately. Do not attempt to work around the system as it may cause data inconsistencies.
