Using the Scanner (Zebra TC27)

Scanner Basics

The TC27 barcode scanner works as a "keyboard wedge" - when you scan a barcode, it types the characters followed by Enter. This means:

  1. Tap the input field to focus it (or it auto-focuses)
  2. Point the scanner at the barcode
  3. Press the yellow scan button
  4. The barcode is automatically entered and submitted
Tip: If scanning doesn't work, make sure the input field has focus (tap it). The cursor should be blinking in the field.

Workflows

1. Receiving Stock

When a shipment arrives at the warehouse:

1

Select Shipment

Go to Receiving and select the pending shipment from the list

2

Verify Contents

Check the products and quantities match the physical delivery

3

Print Case Labels

Generate and print labels for each case. Labels include QR codes for scanning.

4

Apply Labels

Stick each label on its corresponding case

5

Confirm Receipt

Submit the receiving to create stock records

2. Put Away

Moving cases from receiving area to storage locations:

1

Scan Case Label

Scan the QR code on the case you want to put away

2

Scan Destination

Scan the location barcode where you're placing the case

3

Confirm

Review and confirm the put away action

Tip: Put away requires case labels. You must receive stock first to generate case labels.

3. Transfer

Moving stock between warehouse locations:

1

Scan Source Location

Scan the location barcode where the stock currently is

2

Select Stock

If multiple products, select which one to transfer

3

Set Quantity

Choose how many cases to transfer (use +/- buttons)

4

Scan Destination

Scan the location barcode for the destination

5

Confirm

Review the from/to locations and confirm

4. Picking Orders

Picking items for customer orders:

1

Select Pick List

Open an assigned pick list from the picking page

2

Go to Location

Navigate to the displayed location in the warehouse

3

Scan Location

Scan the location barcode to confirm you're at the right spot

4

Pick Items

Take the required quantity and confirm the pick

5

Complete Pick List

After all items picked, mark the pick list as complete

Location Types

TypeDescriptionExamples
Rack Standard pallet racking with multiple levels LOC-A-01-02, LOC-B-05-03
Floor Ground-level storage (no forklift needed) LOC-A-01-00, LOC-A-01-01
Receiving Inbound staging area for new shipments LOC-RECEIVING
Shipping Outbound staging area for dispatch LOC-DISPATCH
Forklift Required: Locations at level 02 and above require a forklift. The system will show a forklift icon for these locations.

Troubleshooting

Scanner Not Working

  • Make sure the input field has focus (tap it)
  • Check the scanner is paired via Bluetooth
  • Try the Scanner Test page to verify scanning works

"Case Not Found" Error

  • Case labels are created during receiving
  • You must receive stock first before doing put away
  • Check if you're scanning the right type of barcode

"Location Not Found" Error

  • Verify the location exists in the system
  • Check the barcode format: LOC-{aisle}-{bay}-{level}
  • Ensure the location is active (not disabled)

"No Stock at Location" Error

  • The scanned location has no stock to transfer
  • Stock may have been moved by another user
  • Check the Stock Overview page to find where stock is located

Tips for Efficiency

  • Batch similar tasks - Do all put aways for one area before moving to another
  • Keep scanner ready - Hold the TC27 in your dominant hand while moving
  • Check dashboard first - See pending put aways and alerts before starting work
  • Use "Scan Next" button - After completing a task, quickly start the next one
  • Print test barcodes - Use /docs/test-barcodes.html for training
