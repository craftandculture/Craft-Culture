Phase 0: Prerequisites Before Testing

Ensure these are ready before starting:

Hardware Setup

  • TC27 Zebra TC27 charged and connected to WiFi
  • TC27 Scanner configured for Keyboard Wedge mode (Enter suffix)
  • ZD421 Printer powered on and paired via Bluetooth or USB
  • ZD421 4" x 6" or 4" x 2" thermal labels loaded

Data Setup

  • At least one Partner exists in the system (for stock ownership)
  • At least one Shipment exists with status ready for receiving
  • Warehouse locations created (or will create in Phase 1)
  • Print test barcodes from /docs/test-barcodes.html

Access

  • Admin account credentials ready
  • Can access https://wine.craftculture.xyz on TC27
  • Review /docs/wms-user-guide.html for workflow reference
