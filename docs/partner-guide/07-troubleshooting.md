# Troubleshooting

Common issues and how to resolve them.

---

## Login Issues

### "Magic link expired"

Magic links expire after 15 minutes.

**Solution:**
1. Go to [craftculture.xyz](https://craftculture.xyz)
2. Click **Sign In**
3. Enter your email again
4. Use the new link immediately

### "Email not found"

Your email isn't registered in our system.

**Solution:**
- Check you're using the correct email
- Contact C&C if you need account setup

### Not receiving login emails

**Check:**
1. Spam/junk folder
2. Correct email address
3. Email filters

**Still not working?** Contact support@craftculture.xyz

---

## Order Issues

### "Cannot submit order"

Common causes:

| Issue | Solution |
|-------|----------|
| Missing client details | Fill all required fields |
| No products added | Add at least one product |
| Missing invoice | Upload partner invoice |
| Validation error | Check error message for details |

### "Order stuck in review"

Orders typically reviewed within 1-3 business days.

**If longer than 3 days:**
1. Check order for any notes from C&C
2. Ensure all documents uploaded
3. Contact C&C for status update

### "Revision requested but I don't see why"

Look for:
1. Notes on the order (from C&C)
2. Email notification with details
3. Activity log on order detail page

---

## Product Issues

### "Can't find a product"

**Try:**
1. Clear search filters
2. Check spelling
3. Search by producer name instead
4. Use partial name

**Still missing?**
- Product may not be in your catalog
- Use "Request Product" to ask C&C

### "Product shows out of stock"

Stock updates automatically from CultX.

**Options:**
1. Check back later
2. Request notification when available
3. Ask C&C about alternatives

### "Price seems wrong"

Prices update with:
- Exchange rate changes
- Supplier price changes

**If incorrect:**
- Compare with your records
- Contact C&C to verify

---

## Payment Issues

### "Payment status not updating"

Payment flow can take time:
- Client payment: 1-7 days
- Distributor payment: 3-7 days
- Partner payment: 1-3 days

**If stuck beyond expected time:**
- Check order detail for notes
- Contact C&C for status

### "Haven't received payment"

Verify:
1. Order status shows "Partner Paid"
2. Payment details on file are correct
3. Check with your bank

**Still missing?** Contact C&C with order number.

---

## Display Issues

### "Page not loading"

**Try:**
1. Refresh the page (Ctrl+R / Cmd+R)
2. Clear browser cache
3. Try different browser
4. Check internet connection

### "Data looks outdated"

**Refresh data:**
- Pull down to refresh (mobile)
- Click refresh icon
- Log out and back in

### "Charts/graphs not showing"

Enable JavaScript in your browser settings.

---

## Mobile Issues

### "App not working on phone"

The platform is web-based, not a native app.

**For best experience:**
1. Use Chrome or Safari
2. Add to home screen for app-like access
3. Ensure latest browser version

### "Text too small"

Pinch to zoom, or use browser accessibility settings.

---

## Error Messages

### "Something went wrong"

General error - try:
1. Refresh page
2. Try again in a few minutes
3. If persists, contact support

### "Session expired"

You've been logged out for security.

**Solution:** Log in again with magic link.

### "Network error"

Check your internet connection and try again.

## Getting Help

### Before Contacting Support

Gather:
- Order number (if applicable)
- Screenshot of error
- Steps to reproduce the issue
- Browser and device info

### Contact Support

**Email:** support@craftculture.xyz

Include:
- Your name and company
- Description of issue
- Order numbers affected
- Screenshots if helpful

**Response time:** Within 24 hours

If the platform seems slow or unavailable:
