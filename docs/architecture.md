## Environment Variables

Stored in Vercel, injected at build/runtime:

| Variable | Purpose |
|----------|---------|
| `DATABASE_URL` | Neon PostgreSQL connection |
| `BETTER_AUTH_SECRET` | Auth encryption key |
| `LOOPS_API_KEY` | Email service |
| `BLOB_READ_WRITE_TOKEN` | Vercel Blob storage |
| `TRIGGER_SECRET_KEY` | Background jobs |
| `GOOGLE_CLIENT_*` | Google Sheets OAuth |

### WMS Workflow Sequence

```
1. RECEIVING
   Shipment arrives â†’ Receive in system â†’ Generate case labels â†’ Print labels

2. PUTAWAY
   Scan case label â†’ Scan destination location â†’ Confirm â†’ Case moved to storage

3. PICKING
   Orders approved â†’ Pick list created â†’ Scan locations â†’ Pick items â†’ Confirm

4. DISPATCH
   Pick list complete â†’ Stage for shipping â†’ Scan dispatch â†’ Mark dispatched
```
